## 安装教程

### 前期准备

开机 F11 进入 BIOS，至少需要做如下修改具体情况还需要看硬件情况：

- VT-d -> Disabled
- XHCI Hand-off -> Enabled

默认使用Opencore引导，如果引导出问题，请更换Clover引导（将EFI-Clover文件夹更名为EFI）

### 机型配置修改：

- OC 自行修改PlatformInfo-Generic里面的参数
- Clover 自行修改SMBIOS和SystemParameters里面的参数

### 当前版本

- Opencore 0.5.3
- Clover 5070
